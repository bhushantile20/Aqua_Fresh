# How to Integrate "Why Choose Us" Section

## Step 1: Add CSS to styles.css

Open `styles.css` and add the CSS code from `why-choose-css.css` at the end of the file.

## Step 2: Add HTML to index.html

Open `index.html` and find this line (around line 268):
```html
        </section>

        <!-- Testimonials Section -->
```

**Insert the HTML from `WHY_CHOOSE_US_HTML.txt` right before the Testimonials Section.**

## Step 3: Verify Font Awesome

Ensure this link is in your HTML `<head>`:
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
```

1. Copy CSS from `why-choose-css.css`
2. Paste into end of `styles.css`
3. Copy HTML from `WHY_CHOOSE_US_HTML.txt`
4. Paste into `index.html` after About section
